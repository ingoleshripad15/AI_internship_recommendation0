# AI_internship_recommendation0
1. GENERAL The AI Based Internship Recommendation System is designed to recommend internships intelligently using Machine Learning.
2. Key idea : • System analyzes skills, domain, and preferences • Uses similarity-based matching • Provides personalized internship suggestions
3.  Problem Identification
   • Lack of personalized internship recommendations based on user profile
   • Difficulty in selecting the right internship from numerous options
   • Time-consuming manual search and filtering process
   • Mismatch between student skills and internship requirements
   • Absence of intelligent systems that analyze user data
   • Limited filtering options based on location, mode (remote/onsite), and duration
   • No proper system to calculate match score or relevance

4. Aims & Objectives
      • Build AI-based recommendation system
      • Provide match score (%) for accuracy
      • Use ML techniques like cosine similarity
      • Enable filtering based on: • Location • Mode (Remote / Onsite / Hybrid)
      • Duration

 5. Scope of Project The system allows:
    • User login & authentication
    • Input of: ◦ Domain ◦ Skills ◦ Preferences (location, mode, duration)
    • AI-based recommendation • Display of: ◦ Internship title
